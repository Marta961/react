# Laboratorio React + TypeScript (Vite)

Proyecto generado con la plantilla **React + TypeScript** de Vite.

## Scripts

```bash
npm install
npm run dev
npm run typecheck
npm run build
```

## Contenido

- `src/components/DataTable.tsx`: tabla genérica `DataTable<T>` con edición usando `Partial<T>`.
- `src/utils/fechas.ts`: diferencia en días con **date-fns** (tipos incluidos en el paquete).
- `docs/arquitectura-final.md`: reflexión sobre genéricos, uniones discriminadas, `never` y utilidades.

## Nota

Este proyecto está pensado para ejecutarse **en local**. No se incluyen pasos de publicación en GitHub.
